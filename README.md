# 🐧 Linux Workshop: Shell Scripting & User Management

A beginner-friendly repository to learn **Linux Shell Scripting** and **User & Group Management** with practical examples.

---

## 📌 About This Repository

This repository contains two core Linux topics:

### 🔹 1. User & Group Management
Learn how to:
- Create and manage users  
- Manage groups  
- Set passwords  
- Control file ownership  
---

### 🔹 2. Shell Scripting Basics
Learn how to:
- Create and run scripts  
- Automate tasks  
- Use loops and commands  
- Build simple real-world scripts
